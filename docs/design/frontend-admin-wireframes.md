# DormMate 관리자 화면 와이어프레임 개요
*작성일: 2025-11-01 · 작성자: Codex(Frontend)*

> Figma 공유가 어려운 환경을 고려해 텍스트/ASCII 기반으로 핵심 화면 레이아웃을 정리했다. 본 문서는 관리자 하단 탭 구조(메인 · 자원 관리 · 권한·계정)의 1차 와이어프레임이다. 상세 스타일은 Tailwind 토큰과 `frontend/components/admin` 공통 컴포넌트를 기준으로 한다.

---

## 1. 메인 탭 (Dashboard)

```
┌────────────────────────────────────────────────────────────┐
│ Header: 로고 · 관리자배지 · 알림벨(outbox) · 프로필       │
├────────────────────────────────────────────────────────────┤
│ Quick KPI Cards (4ea, 2열)                                 │
│ [물품 총계] [임박/만료] [검사 진행률] [알림 실패]         │
├────────────────────────────────────────────────────────────┤
│ ┌──────────────┐  ┌──────────────────────────────────────┐ │
│ │ 층별 상태    │  │ 오늘의 운영 타임라인               │ │
│ │ (Stacked Bar │  │ (검사 완료, 벌점, 예약 등)         │ │
│ │  Chart)      │  │                                    │ │
│ └──────────────┘  └──────────────────────────────────────┘ │
├────────────────────────────────────────────────────────────┤
│ Quick Actions (칸 상태 전환 · 층별장 임명 · 알림 정책 ·  │
│  보고서) → 각각 Drawer/DangerZone 호출                    │
└────────────────────────────────────────────────────────────┘
│ Bottom Nav: [세탁] [냉장고] [도서관] [다목적실] [관리⚙]   │
└────────────────────────────────────────────────────────────┘
```

**상호작용 메모**
- KPI 카드 클릭 시 관련 관리 화면으로 이동 (`FilterBar` 프리셋 적용).
- 타임라인 항목은 `DetailsDrawer`로 연결해 최근 이력·담당자 코멘트 확인.
- 알림 벨은 최근 발송/실패 아웃박스로 직행, `@admin/notifications` Playwright 태그에서 커버.

## 2. 관리(⚙️) > 자원 관리

```
┌────────────────────────────────────────────────────────────┐
│ Header: "자원 관리" · Feature Tag `ADMIN_MODULE_FLAGS`     │
├────────────────────────────────────────────────────────────┤
│ FilterBar                                                   │
│  - 시설 유형 Select (냉장고/세탁/도서관/다목적실)          │
│  - 상태 Segmented (Active/Suspended/Reported/Retired)       │
│  - 검색 Input (라벨·슬롯 코드·보관자·호실)                  │
├────────────────────────────────────────────────────────────┤
│ PaginatedTable (공통)                                       │
│  Columns: 시설 · 위치 · 상태 · 용량/허용량 · 이슈 · 버튼    │
│  Row Action: "상세 보기" → DetailsDrawer                    │
├────────────────────────────────────────────────────────────┤
│ BulkEditor (선택 시 노출)                                   │
│  [선택 해제] [상태 전환] [라벨 범위 조정] [재배분 마법사]   │
├────────────────────────────────────────────────────────────┤
│ DetailsDrawer (우측)                                        │
│  - 기본 정보 (상태 토글, 담당자, 메모)                      │
│  - 탭1: 칸/슬롯 리스트 + 재배분 버튼                        │
│  - 탭2: 최근 검사 이력                                       │
│  - DangerZone: 칸 삭제 / 데모 Seed                           │
└────────────────────────────────────────────────────────────┘
```

**상호작용 메모**
- `재배분 마법사`는 Drawer 내부 Stepper(1. 현재 요약 → 2. 새 배정 → 3. 검증)를 활용.
- DangerZoneModal은 prod 빌드에서 비활성화(Feature Flag 필요).
- 테이블 선택 상태는 `BulkEditor`가 관리하며 Playwright @admin-resource 시나리오에서 검증.
- 삭제 이력이 있는 포장은 목록에서 `라벨 재사용` 배지로 강조하고, Hover 시 최근 삭제 시각을 Tooltip으로 보여 준다.

## 3. 관리(⚙️) > 권한·계정

```
┌────────────────────────────────────────────────────────────┐
│ FilterBar                                                   │
│  - 검색 (이름/호실)                                         │
│  - 역할 Segmented (거주자/층별장/관리자)                   │
│  - 상태 Select (활성/비활성)                                │
├────────────────────────────────────────────────────────────┤
│ PaginatedTable                                              │
│  Columns: 이름 · 호실 · 역할배지 · 최근 로그인 · 액션       │
│  Row Action: [상세] → DetailsDrawer                         │
├────────────────────────────────────────────────────────────┤
│ DetailsDrawer                                               │
│  - 역할 토글 (Radio)                                        │
│  - 진행 중 검사 세션 안내 (있을 경우 Highlight)             │
│  - 벌점 요약 및 링크                                        │
├────────────────────────────────────────────────────────────┤
│ Footer Quick Actions                                        │
│  [층별장 임명] [층별장 해제] [관리자 임명]                  │
│   → DangerZoneModal (사유 입력 필수)                        │
└────────────────────────────────────────────────────────────┘
```

**상호작용 메모**
- 다중 선택 시 BulkEditor는 `층별장 해제` 등의 버튼만 노출.
- 임명/해제 시 감사 로그 및 알림 발송 여부를 Drawer 하단에서 선택.
- Playwright @admin-roles 태그로 층별장 승격/복귀 흐름 자동화.

---

## 4. 공통 반응형 가이드
- **모바일(≤768px)**: FilterBar는 2열 그리드 → 아코디언, 테이블은 카드뷰 전환(핵심 필드만 표시).
- **태블릿**: DetailsDrawer는 하단 Sheet 모드, BulkEditor는 테이블 하단 고정.
- **데스크톱**: Drawer 우측 고정, BulkEditor 상단 혹은 하단 배치 선택 가능.
